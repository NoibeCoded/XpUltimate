## About The Project

XpUltimate is the ultimate experience plugin! It's a rework of CodedRed's XpBottles concept, but with support removed for outdated versions, updated to the latest version for continued maintenance. XpUltimate also introduces a new XpBank feature. Create experience bottles, add, remove, or reset player experience, save experience in the XP Bank, transfer it to other users, and much more!
Experience is stored in a hidden location, allowing you to customize the item's history and name as you wish. No restrictions.
